# Forkchoice Frontend

Forkchoice bundled frontend

## Get started

```
npm install

npm run dev
```

### Building

```
npm run build
```

### Storybook

```
npm run storybook
```

### Tests

```
npm run tests
```

## Upgrading

### Storybook

Currently running storybook 7.0 rc for vite first class support.

```bash
npx sb@next upgrade --prerelease
```
